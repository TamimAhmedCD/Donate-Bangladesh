Important FAQ

What is the Document object Model (DOM)?

The Document Object Model (DOM) is a hierarchical structure that allows scripts to access and manipulate the content and structure of web documents.

How do select an elements from the DOM?

JavaScript provides several ways to select elements from the Document Object Model (DOM), including using an ID, a CSS selector, or a tag name

what is event delegation in the context of DOM, and why is useful?

Event delegation is a technique where a single event listener on a parent element handles events for all its child elements, improving performance and simplifying code.

how do you manipulate an element's attributes and style using the DOM?

You can manipulate an element's attributes using element.setAttribute('attrName', 'value') and its style using element.style.propertyName = 'value'.
